## 柯南主题——力学拖拽视图demo(移动端&PC端)

* [预览链接](http://123.57.24.197/drag)

项目介绍：<br>
1、基于d3.js开发的柯南视图，能够自由拖拽、缩放，可用于移动端的思维导图；<br>
2、通过导入不同的关系数据生成相应的力学导图；<br>
